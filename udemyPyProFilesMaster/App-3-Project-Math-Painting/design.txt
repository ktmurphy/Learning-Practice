Title: Math Painting
Description: An app that lets the user provide the start coordinates of geometrical shapes such as
squares and rectangles, their dimensions, and their colors, and the program produces an image file canvas with all
the geometrical shapes drawn in it.
Objects: Square:
            x
            y
            side
            color
            draw(canvas)
         Rectangle:
            x
            y
            width
            height
            color
            draw(canvas)
         Canvas:
            width
            height
            color
            make(imagepath)



Title: Math Painter
Description: A command-line app that lets you draw rectangles and squares 
on a black or white canvas by indicating the type of shape, the 
top right corner, the dimensions, and the color. 

Objects:

Rectangle: 
   x coordinate
   y coordinate
   width
   height
   color
   Draw(canvas)
Square: 
   x coordinate
   y coordinate
   side length
   color
   Draw(canvas)
Canvas: 
   width
   height
   color
   make(imagePath)
